Questions:

  1. What does the server in the client server architecture do?

    The client server architecture is a network of computers where clients are responsible to request services and the servers are responsible to provide the services asked for.

    This way all information can be stored away on other host computers, however still accessable by other user computers.

  2. What does the client do?

    The cliens responsiblity is to provide an interface where the users can ask for a website and the website is requested by the client from the server.

    Then the user will obtain a display of the website or service that the user asked for.

  3. What is an IP address?

    An IP adress is an adress or a link to another computer. All computers has it's own IP, and what you are simply calling for when

    you are asking for a different ip is just another computer. Which in most cases are servers which is responsible for storing information.

    Such as website interfaces or data.

  4. What is DNS?

    DNS stands for domain name service, and is responsible for connecting domain name with the ip adress.

    It works as a database which is spread over multiple computers, responsible to store the domain name and the equivalent ip adress.

    Thereby accesing it when the client asks for it.

  5. What is the difference between the Internet and WWW?

    The internet is a globally conected network which is able to communicate with each other for data and rescoures around the globe. It is connected by

    a collection of networks. Both private, public, buisinesses etc. Which togheter makes it easy to access data and communicate with each other around the world.

    World wide web (WWW) however, is the documents and websites you access every day. These websites are connected by hypertext, which allows people to access documents

    by just writing a phrase from it.

  6. Explain what HTML is

    HTML stands for HyperText Markup Language, which is the official langauge used to write these "documents" which we talked about when answering last question.

    The language is made to structurize these docuemnts into proper websites which can be accessed through the world wide web. In other words, a

    coding language to make it easier for people to set up pages.
